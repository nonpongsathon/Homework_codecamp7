2.Differntiation tag

tag Similarity Difference
hr & br เป็นการแบ่งระหว่าง content ข้างบนกับข้างล่าง br แบ่งแบบเว้นช่องว่าง ส่วน hr แบ่งแบบขีดเส้นกั้น
ul & ol เป็นการ list item เหมือนกัน ul ใช้ index เป็น bullet point ส่วน ol ใช้เป็นตัวเลข
select & checkbox & radio button เป็นการเลือกแบบมีตัวเลือก select มีลูกศร กดเลื่อนลงมา
checkbox เป็นกล่องสี่เหลี่ยมให้ติ๊ก
radio เป็นวงกลม ให้เลือกจุด
div & span เป็นกล่องข้อความเหมือนกัน div แสดงกล่องข้อความในบรรทัดถัดไป
span แสดงกล่องข้อความในบรรทัดเดิม
script & style ใช้ในการตกแต่ง จาก CSS และ Javascript style มักวางไว้ในส่วน header
script มักวางไว้ในส่วน footer
