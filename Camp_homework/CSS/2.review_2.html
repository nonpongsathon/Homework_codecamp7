Property vs Value Property มีกี่กลุ่ม Value unit มีกี่ประเภท อะไรบ้าง แต่ละประเภทคืออะไร

Property คือ คุณสมบัติที่ต้องการกำหนด

Value คือ ค่าที่ต้องการกำหนด ตามคุณสมบัตินั้นๆ

 Property มีทั้งหมด 5 กลุ่ม
  • Content
  • Border
  • Padding
  • Margin
  • Element
 Unit มี 2 ประเภท คือ
  1. Absolute lengths = unit ที่มีค่าของตัวเอง
  2. Relative lengths = unit ที่มีค่าอ้างอิงตาม element อื่นๆ

For example Property and value

color:red

width:200px

padding:10px

Example for property

  1. content - color, size, font, text
  2. border - border, table
  3. padding - padding(top, right, bottom, left)
  4. margin - margin(top, right, bottom, left)
  5. elememt - display, height, width, position, float, flex, grid

Unit Examples

Absolute - cm, mm, in, px, pt, pc

Relative - em, ex, vh, vw, %
