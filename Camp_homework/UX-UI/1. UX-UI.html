เว็บไซต์ที่เลือกมาได้แก่ Careerjet



  • Useful : มีประโยชน์สำหรับผู้ที่ต้องการค้นหางาน และสมัคร
  • Usable : ใช้งานง่ายเนื่องจาก มีโครงสร้างที่ไม่ซับซ้อนและเมนูไม่เยอะ
  • Findable : หาสิ่งที่ต้องการได้ง่าย มีแถบ search หมวดหมู่ รวมถึงสิ่งที่เราสนใจจะแสดงขึ้นหน้าแรก
  • Credible : ไม่ค่อยน่าเชื่อถือเท่าไร เนื่องจากไม่ได้เป็นที่รู้จักมากนัก
  • Desirable : ไม่ค่อยน่าใช้ เนื่องจากเป็นเว็บที่ไม่มีการตกแต่งใดๆ ไม่ทำให้เกิด First Impression
  • Accessible : เข้าถึงได้ง่ายในแง่ของการเข้ามาใช้งานฟรี ไม่ว่าใครที่มีอินเตอร์เน็ตกับอุปกรณ์ที่ต่ออินเตอร์เน็ตได้ ก็เข้าถึงได้
