position คืออะไร มีอะไรบ้าง z-index คืออะไรใช้อย่างไร
position คือการกำหนดตำแหน่งของ element โดยมีอยู่ 5 รูปแบบ ได้แก่ static, relative, fixed. absolute, sticky z-index คือ การกำหนดลำดับก่อนหลังของ element ในมุมมองแกน z โดยวิธีใช้คือกำหนดตัวเลขลำดับให้กับ element นั้นๆ ตัวเลขมากกว่าอยู่ข้างบน น้อยกว่าอยู่ข้างล่าง
