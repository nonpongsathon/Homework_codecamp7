Value ใน property grid และ flex ต่างกันอย่างไร justify และ align ต่างกันอย่างไร
  • inline อยู่บรรทัดเดียวกัน เหมือน span
  • block อยู่คนละบรรทัดเหมือน p
  • contents ทำให้ element เป็น contents ปรับ margin,padding,border,background ไม่ได้
  • flex จัดรูปแบบ flex
  • grid จัดรูปแบบ grid
  • inline-(block,flex,grid,table)
  • run-in ขึ้นอยู่กับเนื้อหา
  • none ไม่แสดงผล
 - flex ใช้ในการจัด layout ที่มี 1 มิติ (แนวตั้งหรือแนวนอน) และ grid ใช้เพื่อจัด layout ที่มี 2 มิติ (มีทั้งแนวตั้งและแนวนอน) - justify-content ใช้กำหนด value ของ flex ในการจัดแนวนอน และใช้เหมือน align-content(flex) หรือกำหนด value ของ grid ในแนวนอน ใช้เหมือนกับ align-content (grid)
-align ใช้กับ flex กำหนดการจัดแนวตั้ง
